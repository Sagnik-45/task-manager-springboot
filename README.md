# Task Manager Application

A full-stack task manager built with Spring Boot and MySQL.

## Tech Stack
- Spring Boot
- MySQL
- JPA / Hibernate
- HTML
- CSS
- JavaScript

## Features
- Create tasks
- Update tasks
- Delete tasks
- Validation
- Exception handling

## API Endpoints

GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/{id}  
DELETE /api/tasks/{id}
